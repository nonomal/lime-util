## 浏览器 Device

#### \_.trimAll(str)

去掉所有的空格

- ##### 参数

`str` String|Number 参数

- ##### 返回值

String 返回处理后的字符串

- ##### 示例

```javascript
import LimeUtil from "lime-util";
let res = LimeUtil.trimAll("a b c ");
console.log(res); // 输出 abc
```

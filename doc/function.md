## 函数

#### \_.trimAll(str)

去掉所有的空格

- ##### 参数

`str` String|Number 参数

- ##### 返回值

String 返回处理后的字符串

- ##### 示例

```javascript
import LemonUtils from "lemon-utils";
let res = LemonUtils.trimAll("a b c ");
console.log(res); // 输出 abc
```
